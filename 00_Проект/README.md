# 00_Проект

Раздел управления Knowledge Factory «Теплоконтур».

## Состав
- `01_Управление` — учредительные и регламентирующие документы (КП-001…КП-007).
- `02_Реестр` — реестр документов, реестр версий, реестр терминов.
- `03_Sprint_Report` — отчёты по спринтам разработки базы знаний.
- `MANIFEST.xlsx` — сводная таблица состояния всех разделов Проекта на текущий релиз.
- `CHANGELOG.md` — история изменений по релизам.

## С чего начать
1. Прочитать `01_Управление/КП-001_Конституция_проекта.docx`.
2. Ознакомиться с жизненным циклом документа — `01_Управление/КП-002_Регламент_разработки.docx`.
3. Использовать шаблоны из `../17_Шаблоны` для создания новых документов.
4. Актуальный статус всех разделов — в `MANIFEST.xlsx`.
