# 02_Регламенты_РГЛ

Регламенты обслуживания оборудования.

## Наполнение
- **РГЛ-001** — «Регламент планового обслуживания теплообменных аппаратов», v1.0 (сформирован в Режиме Б по открытым данным, смежный документ — СТД-001).

## Как добавить новый регламент
Создавать новые документы по шаблону 17_Шаблоны/ШБ-002_РГЛ.docx, регистрируя их в 00_Проект/02_Реестр/Реестр_документов.xlsx (см. КП-002). Для формирования черновика можно использовать промпты 07_Промпты/02_РГЛ (Режим А — если есть данные объекта, Режим Б — автономный подбор по открытым данным).
