# 02_Регламенты_РГЛ

Регламенты обслуживания оборудования.

## Наполнение
- **РГЛ-001** — «Регламент планового обслуживания теплообменных аппаратов», v1.1 (смежный документ — СТД-001).
- **РГЛ-002** — «Регламент планового обслуживания водогрейных котлов», v1.0 (смежный документ — СТД-002).
- **РГЛ-003** — «Регламент планового обслуживания циркуляционных насосов», v1.0 (смежные документы — СТД-003, ИНС-001).

Теперь для всех трёх типов оборудования, охваченных стандартами диагностики (01_Стандарты_СТД), есть парный регламент обслуживания.

## Как добавить новый регламент
Создавать новые документы по шаблону 17_Шаблоны/ШБ-002_РГЛ.docx, регистрируя их в 00_Проект/02_Реестр/Реестр_документов.xlsx (см. КП-002). Для формирования черновика можно использовать промпты 07_Промпты/02_РГЛ (Режим А — если есть данные объекта, Режим Б — автономный подбор по открытым данным).
