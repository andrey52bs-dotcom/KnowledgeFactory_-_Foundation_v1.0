# 03_Инструкции_ИНС

Пошаговые инструкции для персонала.

## Наполнение
- **ИНС-001** — «Порядок запуска циркуляционного насоса», v1.0, статус «Утверждено». Раздел «Типичные ошибки» составлен на основании 6 документированных отраслевых случаев из открытых источников (реальные задокументированные инциденты, не гипотетические примеры) — закрытая напорная задвижка при пуске, непроверенное положение арматуры, запуск без удаления воздуха, вибрация принятая за механическую неисправность, обманчиво нормальные показания приборов, дистанционный запуск без обхода оборудования.
