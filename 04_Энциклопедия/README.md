# 04_Энциклопедия

Теоретические материалы серии ЭНЦ.

## Наполнение
- **ЭНЦ-001** — Теплопередача в теплообменных аппаратах (теория для СТД-001, РГЛ-001)
- **ЭНЦ-002** — Процесс горения и КПД водогрейных котлов (теория для СТД-002)
- **ЭНЦ-003** — Принципы работы центробежных насосов (теория для СТД-003, ИНС-001)

Каждая статья объясняет физический механизм, стоящий за конкретными диагностическими правилами связанного СТД — не абстрактная теория, а обоснование того, почему стандарт диагностики устроен именно так.
