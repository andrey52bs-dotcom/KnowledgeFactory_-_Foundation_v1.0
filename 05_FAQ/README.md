### Для клиентов (`Клиенты/`)

Статус — **«На редакции»**: требуют проверки ответственным лицом перед публикацией на сайте (особенно FAQ-К-003 про запах газа — вопрос безопасности). В нескольких местах оставлены плейсхолдеры `[УТОЧНИТЬ: ...]` для контактов диспетчерской службы — их нужно заполнить реальными данными компании.
