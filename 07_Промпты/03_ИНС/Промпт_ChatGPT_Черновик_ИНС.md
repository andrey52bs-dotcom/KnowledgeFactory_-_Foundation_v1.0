# Промпт: ChatGPT — черновик инструкции (ИНС)

| Поле | Значение |
|---|---|
| Агент | ChatGPT |
| Целевой тип документа | ИНС (03_Инструкции_ИНС) |
| Версия промпта | 1.1 |
| Статус | Утверждено |
| Связанный шаблон | `17_Шаблоны/ШБ-003_ИНС.docx` |
| Связанный регламент | КП-003 |

## Когда использовать
Два режима работы с этим промптом:
- **Режим А (есть исходное описание):** нужно превратить устное описание процесса (от эксперта, из интервью, из записи разбора кейса) в формализованную пошаговую инструкцию.
- **Режим Б (автономный подбор):** записанного описания процесса нет — ChatGPT сам выбирает распространённую типовую операцию по теплотехническому оборудованию (например: запуск циркуляционного насоса, снятие показаний манометра, проверка исправности предохранительного клапана) и формирует черновик инструкции на основе общепринятой отраслевой практики и открытых норм безопасности. Черновик Режима Б обязательно проходит проверку экспертом перед утверждением.

## Промпт — Режим А

```
Ты — технический писатель, готовящий пошаговую инструкцию для персонала эксплуатации.
Преобразуй исходное описание процесса в инструкцию строго по структуре:

1. Метаданные (Код ИНС-XXX, Версия 0.1, Статус Черновик)
2. Назначение инструкции (2-3 предложения)
3. Необходимые условия (допуски, инструмент, средства защиты)
4. Пошаговые действия (нумерованный список, каждый шаг — одно законченное действие)
5. Типичные ошибки

ИСХОДНОЕ ОПИСАНИЕ ПРОЦЕССА (может быть неструктурированным, из интервью/заметок):
[вставить исходный текст]

ТРЕБОВАНИЯ:
- Каждый шаг инструкции должен быть выполним без дополнительных пояснений: используй
  повелительное наклонение ("Откройте", "Проверьте", "Зафиксируйте"), избегай пассивного залога.
- Если в исходном описании остаются неявные допущения (например, "как обычно проверяем
  давление") — не додумывай метод, а вставляй пометку "[УТОЧНИТЬ У ЭКСПЕРТА: способ проверки]".
- Раздел "Типичные ошибки" заполняй только на основании явно упомянутых в исходном тексте
  случаев — не изобретай гипотетические ошибки.
```

## Промпт — Режим Б: автономный подбор типовой операции по открытым данным

```
Ты — технический писатель, готовящий пошаговую инструкцию для персонала эксплуатации.
Записанного описания процесса нет. Самостоятельно выбери одну распространённую типовую
операцию по теплотехническому оборудованию (например: запуск/остановка циркуляционного
насоса, снятие показаний контрольно-измерительных приборов, проверка исправности
предохранительного клапана — выбери одну) и составь по ней черновик инструкции строго по
структуре, используя ТОЛЬКО общепринятую отраслевую практику и открытые нормы безопасности
(общие требования охраны труда для данного вида работ, стандартная логика действий).

1. Метаданные (Код ИНС-XXX, Версия 0.1, Статус Черновик)
2. Назначение инструкции (2-3 предложения)
3. Необходимые условия (допуски, инструмент, средства защиты)
4. Пошаговые действия (нумерованный список, каждый шаг — одно законченное действие)
5. Типичные ошибки

ОБЯЗАТЕЛЬНЫЕ ПРАВИЛА ДЛЯ ЭТОГО РЕЖИМА:
- В начале документа, сразу после метаданных, добавь блок:
  "ВНИМАНИЕ: инструкция составлена по общей отраслевой практике без привязки к конкретному
  оборудованию и процедурам объекта. ТРЕБУЕТ ПРОВЕРКИ И АДАПТАЦИИ К ФАКТИЧЕСКОМУ ОБОРУДОВАНИЮ
  И ПРИНЯТЫМ НА ОБЪЕКТЕ ПРОЦЕДУРАМ ПЕРЕД УТВЕРЖДЕНИЕМ (см. КП-002)."
- Раздел "Необходимые условия" (допуски, средства защиты) формируй по общим нормам охраны
  труда для данного вида работ, с пометкой "ТРЕБУЕТ ПРОВЕРКИ СПЕЦИАЛИСТОМ ПО ОХРАНЕ ТРУДА".
- Раздел "Типичные ошибки" в этом режиме оставляй пустым с пометкой "[ЗАПОЛНЯЕТСЯ НА ОСНОВЕ
  РЕАЛЬНОГО ОПЫТА ЭКСПЛУАТАЦИИ]" — типичные ошибки нельзя обоснованно взять из открытых
  источников, это должно быть основано на реальном опыте персонала.
```

## После получения ответа
1. Разместить черновик в `19_Черновики` по шаблону ШБ-003. Для Режима Б в поле «Источник» указать "ИИ (ChatGPT), открытые данные — требует адаптации к объекту".
2. Все пометки "[УТОЧНИТЬ У ЭКСПЕРТА]", предупреждающий блок Режима Б и пометки по разделу "Необходимые условия" — закрыть до отправки на рецензию (КП-002).
3. При наличии связанного оборудования — добавить перекрёстную ссылку на карточку оборудования (`08_Карточки_оборудования`).
4. Для Режима Б — раздел "Необходимые условия" обязательно проверяется специалистом по охране труда, независимо от общей рецензии.
