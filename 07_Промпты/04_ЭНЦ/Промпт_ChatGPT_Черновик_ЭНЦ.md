# Промпт: ChatGPT — черновик энциклопедической статьи (ЭНЦ)

| Поле | Значение |
|---|---|
| Агент | ChatGPT |
| Целевой тип документа | ЭНЦ (04_Энциклопедия) |
| Версия промпта | 1.0 |
| Статус | Утверждено |
| Связанный шаблон | `17_Шаблоны/ШБ-004_ЭНЦ.docx` |
| Связанный регламент | КП-003 |

## Когда использовать
Когда в СТД, РГЛ или ИНС встречается физическое явление или понятие, требующее отдельного теоретического объяснения (раздел 11_Физика_процессов / 04_Энциклопедия), и нужно быстро подготовить первую версию статьи.

## Промпт

```
Ты — автор технической энциклопедии по эксплуатации теплотехнического оборудования.
Подготовь энциклопедическую статью по структуре:

1. Метаданные (Код ЭНЦ-XXX, Версия 0.1)
2. Понятие / явление — определение и физический смысл
3. Практическое значение — как это знание применяется при диагностике и обслуживании
4. Связанные разделы (укажи конкретные документы, если они известны, например СТД-001)

ПОНЯТИЕ / ЯВЛЕНИЕ: [указать, например "теплопередача в кожухотрубном теплообменнике"]
КОНТЕКСТ ПРИМЕНЕНИЯ (в каком СТД/РГЛ встретилось): [указать]

ТРЕБОВАНИЯ:
- Раздел "Понятие / явление" пиши на уровне инженера-эксплуатационщика, а не физика-теоретика:
  без вывода формул, но с точным физическим механизмом.
- Раздел "Практическое значение" обязателен и не может быть общим ("это важно для диагностики") —
  укажи, на какой конкретно контролируемый параметр или признак неисправности это влияет.
- Не используй термины, отсутствующие в 09_Технический_словарь, без пояснения в скобках при
  первом употреблении.
```

## После получения ответа
1. Разместить черновик в `19_Черновики` по шаблону ШБ-004.
2. Указать явную двустороннюю ссылку: в статье ЭНЦ — на СТД/РГЛ, где применяется, и в этом СТД/РГЛ (раздел «Ссылки») — на новую статью ЭНЦ.
