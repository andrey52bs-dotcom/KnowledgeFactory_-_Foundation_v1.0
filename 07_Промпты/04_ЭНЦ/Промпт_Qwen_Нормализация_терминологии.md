# Промпт: Qwen — нормализация терминологии

| Поле | Значение |
|---|---|
| Агент | Qwen |
| Целевой раздел | 09_Технический_словарь (применяется ко всем типам документов) |
| Версия промпта | 1.0 |
| Статус | Утверждено |
| Связанный регламент | КП-003, КП-007 |

## Когда использовать
При добавлении нового документа любого типа (СТД, РГЛ, ИНС, ЭНЦ) — чтобы проверить, что используемая в нём терминология совпадает с уже зафиксированной в `Реестр_терминов.xlsx`, и выявить синонимы / несогласованные варианты написания одного и того же понятия.

## Промпт

```
Ты выполняешь нормализацию технической терминологии для корпоративной базы знаний.
На вход подаётся новый документ и текущий реестр терминов.

Выполни:
1. Извлеки из документа все специальные технические термины (существительные и устойчивые
   словосочетания, относящиеся к оборудованию, процессам, параметрам диагностики).
2. Для каждого термина сверь с реестром терминов:
   - Если термин уже есть в реестре под другим написанием (синоним, сокращение, иная форма
     слова) — укажи оба варианта и предложи, какой считать основным (по частоте использования
     в уже утверждённых документах реестра).
   - Если термина в реестре нет — добавь его в список "Кандидаты на добавление в реестр"
     с кратким определением на основе контекста документа.
3. Не смешивай русскую и калькированную с английского терминологию без необходимости —
   если в документе встречается англицизм при наличии устоявшегося русского термина в реестре,
   отметь как несоответствие.

Результат оформи в виде двух списков: "Расхождения с реестром" и "Кандидаты на добавление".
Не редактируй сам документ — только анализ терминологии.

ВХОДНЫЕ ДАННЫЕ:
Документ: [вставить текст]
Текущий реестр терминов (выгрузка из Реестр_терминов.xlsx): [вставить]
```

## После получения ответа
1. Расхождения — исправить в документе до утверждения (КП-002).
2. Кандидатов на добавление — согласовать с ответственным за раздел 09_Технический_словарь и внести в `Реестр_терминов.xlsx` одновременно с публикацией документа (КП-002, раздел «Требования к содержанию»).
