# Промпт: Gemini — анализ фото/видео оборудования для кейса или каталога неисправностей

| Поле | Значение |
|---|---|
| Агент | Gemini |
| Область применения | 13_Кейсы_объектов, 10_Каталог_неисправностей, 14_Фотоархив, 15_Видеоархив |
| Версия промпта | 1.0 |
| Статус | Утверждено |
| Связанный шаблон | `17_Шаблоны/ШБ-006_КЕЙС.docx` |
| Связанный регламент | КП-003 |

## Когда использовать
Когда с объекта поступили фото или видео неисправности/осмотра оборудования, и нужно быстро получить структурированное текстовое описание для оформления кейса или пополнения каталога неисправностей.

## Промпт

```
Ты анализируешь фото- или видеоматериал по осмотру теплотехнического оборудования.
Опиши ТОЛЬКО то, что реально видно на изображении/видео — не делай диагностических выводов
о причине неисправности, если она не видна напрямую (например, не пиши "проблема в клапане",
если клапан не виден на кадре).

Структурируй ответ:
1. Тип оборудования (если определим по внешнему виду)
2. Видимые признаки (внешний вид, следы протечек, коррозии, накипи, повреждений — перечисли
   всё замеченное, включая мелкие детали)
3. Расположение признака на оборудовании (например, "в районе фланцевого соединения на входном
   патрубке")
4. Что НЕЛЬЗЯ определить по данному материалу (явно перечисли ограничения — например,
   "невозможно оценить внутреннее состояние без вскрытия")

Не предполагай марку/модель оборудования, если она не читается на шильдике или бирке в кадре.

ВХОДНОЙ МАТЕРИАЛ: [прикрепить фото/видео]
КОНТЕКСТ (если известен): [объект, дата, кто снимал]
```

## После получения ответа
1. Текстовое описание используется как раздел «Проявление неисправности» кейса по шаблону ШБ-006 — дополняется человеком-экспертом диагностическими выводами и решением.
2. Сам материал (фото/видео) сохраняется в `14_Фотоархив` / `15_Видеоархив` со ссылкой на кейс.
3. Признак с описанием — кандидат на добавление в `10_Каталог_неисправностей`, если он ранее не зафиксирован.
