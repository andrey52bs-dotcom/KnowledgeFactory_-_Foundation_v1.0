# 08_Карточки_оборудования

Паспортные и эксплуатационные карточки оборудования, на которые ссылаются стандарты диагностики (01_Стандарты_СТД) и регламенты (02_Регламенты_РГЛ).

## Наполнение
Шаблон **ШБ-009_Карточка_оборудования.docx** (17_Шаблоны) + 3 примера заполнения по уже готовым СТД:

| Код | Оборудование | Связанные документы |
|---|---|---|
| КАР-001 | Теплообменный аппарат (пример) | СТД-001, РГЛ-001, ЭНЦ-001 |
| КАР-002 | Водогрейный котёл (пример) | СТД-002, ЭНЦ-002 |
| КАР-003 | Циркуляционный насос (пример) | СТД-003, ИНС-001, ЭНЦ-003 |

**Важно:** все три карточки — это **примеры структуры**, а не паспорта реальных единиц оборудования объекта. Паспортные значения — типовые, инвентарные номера, серийные номера и реальные параметры отмечены плейсхолдерами `[ЗАПОЛНЯЕТСЯ ПРИ ВВОДЕ РЕАЛЬНОЙ ЕДИНИЦЫ]`. Раздел «Журнал диагностики и обслуживания» в примерах пуст — он предназначен для накопления реальной истории эксплуатации конкретной единицы.

## Как завести карточку реальной единицы оборудования
1. Скопировать шаблон `ШБ-009_Карточка_оборудования.docx` (или один из примеров КАР-001/002/003, если тип оборудования совпадает).
2. Заполнить реальные метаданные и паспортные характеристики.
3. Присвоить следующий свободный код КАР-XXX и зарегистрировать в `Реестр_документов.xlsx`.
