# 09_Технический_словарь

Единая терминология Проекта.

## Наполнение
**Технический_словарь.docx** — читаемый, отсортированный по алфавиту документ на 29 терминов: 14 терминов, встречающихся в СТД-001/002/003, ЭНЦ-001/002/003, РГЛ-001 и др., плюс 15 терминов, сверенных и дополненных из внешних нормативных источников (ФЗ №190-ФЗ «О теплоснабжении», ГОСТ 22270-2018, СП 124.13330.2012) — см. `16_Нормативные_документы/НД-001`.

**Важно (единственный источник истины, КП-001):** мастер-источник терминологии — `00_Проект/02_Реестр/Реестр_терминов.xlsx`. Документ в этой папке — производное читаемое представление реестра, а не отдельный источник. Новый термин сначала добавляется в реестр (КП-002, КП-007), затем этот документ обновляется вслед за ним.

## Как добавить новый термин
1. Добавить строку в `Реестр_терминов.xlsx` при публикации документа, где термин впервые встречается (КП-002, раздел «Требования к содержанию»).
2. Обновить `Технический_словарь.docx`, чтобы он оставался актуальным читаемым представлением реестра.
