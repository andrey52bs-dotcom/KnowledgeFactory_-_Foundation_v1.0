# 10_Каталог_неисправностей

Классификатор неисправностей и их признаков.

## Наполнение
**Каталог_неисправностей.xlsx** — 7 неисправностей, все ранее уже упоминались по названию в СТД-001/002/003, ЭНЦ-001/002/003 и RAG-карточках, но не были собраны в едином месте. Этот файл закрывает эти ссылки:

| Код | Неисправность | Оборудование |
|---|---|---|
| НСП-001 | Загрязнение теплообменной поверхности | Теплообменные аппараты |
| НСП-002 | Загрязнение поверхностей нагрева | Водогрейные котлы |
| НСП-003 | Неполнота сгорания топлива | Водогрейные котлы |
| НСП-004 | Износ рабочего колеса | Циркуляционные насосы |
| НСП-005 | Кавитация | Циркуляционные насосы |
| НСП-006 | Износ подшипников | Циркуляционные насосы |
| НСП-007 | Дисбаланс ротора | Циркуляционные насосы |

Колонка «Типовое решение» для всех строк пока содержит только текстовую отсылку — раздел `12_Типовые_решения` ещё не наполнен. Когда он появится, эту колонку нужно будет заменить точными ссылками на конкретные документы.
