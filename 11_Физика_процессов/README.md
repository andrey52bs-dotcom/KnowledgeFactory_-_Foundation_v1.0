# 11_Физика_процессов

Физические принципы, лежащие в основе диагностики оборудования.

## Важное примечание (единственный источник истины, КП-001)
Теоретические статьи по физическим механизмам, объясняющим диагностические правила стандартов (СТД), фактически размещены в разделе **`04_Энциклопедия`** (серия ЭНЦ) — там уже утверждены ЭНЦ-001 (теплопередача), ЭНЦ-002 (горение и КПД), ЭНЦ-003 (центробежные насосы), каждая с прямой ссылкой на соответствующий СТД.

Ранее СТД-001/002/003 содержали двойную ссылку — одновременно на `11_Физика_процессов` и на конкретную статью ЭНЦ, что нарушало принцип «единственный источник истины» (КП-001, раздел 4). Дублирующая ссылка удалена из СТД-001/002/003 (→ версия 1.2) — теперь ссылка ведёт только на конкретную статью ЭНЦ.

Этот раздел (`11_Физика_процессов`) остаётся зарезервированным для будущих материалов, которые по своему характеру шире серии ЭНЦ (например, сводные теоретические курсы, не привязанные к одному конкретному СТД). До появления такого материала — актуальную теорию искать в `04_Энциклопедия`.
