# 12_Типовые_решения

Готовые решения для типовых проблем, выявленных в ходе диагностики и обслуживания оборудования.

## Наполнение
**Типовые_решения.xlsx** — 7 решений, каждое устраняет конкретную неисправность из `10_Каталог_неисправностей` (соответствие 1:1):

| Код | Решение | Устраняет |
|---|---|---|
| РЕШ-001 | Промывка теплообменных пластин/трубок | НСП-001 |
| РЕШ-002 | Очистка поверхностей нагрева котла | НСП-002 |
| РЕШ-003 | Наладка режима горения | НСП-003 |
| РЕШ-004 | Замена рабочего колеса насоса | НСП-004 |
| РЕШ-005 | Устранение условий кавитации | НСП-005 |
| РЕШ-006 | Замена подшипников насоса | НСП-006 |
| РЕШ-007 | Балансировка ротора насоса | НСП-007 |

Колонка «Типовое решение» в `10_Каталог_неисправностей/Каталог_неисправностей.xlsx` обновлена точными ссылками на эти коды.
