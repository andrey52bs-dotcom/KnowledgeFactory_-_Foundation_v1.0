# 16_Нормативные_документы

Внешние нормативные документы и стандарты, на которые ссылаются внутренние документы Проекта.

## Наполнение
**НД-001_Индекс_нормативных_документов.docx** — индекс 3 внешних документов, чья терминология была сверена и добавлена в проект:

| Документ | Использовано для |
|---|---|
| ФЗ №190-ФЗ «О теплоснабжении» | Базовая терминология (тепловая энергия, теплоноситель, тепловая сеть, система теплоснабжения) |
| ГОСТ 22270-2018 «Системы ОВК. Термины и определения» | Терминология оборудования (теплообменник, водяное отопление, бак-аккумулятор) |
| СП 124.13330.2012 «Тепловые сети» | Терминология тепловых пунктов (тепловой пункт, ИТП, ЦТП, АУУ) |

**Важно:** полный текст этих документов НЕ воспроизводится (авторское право издающих органов) — только атрибутированные термины в `09_Технический_словарь` и ссылка на официальный источник для получения актуального текста.
