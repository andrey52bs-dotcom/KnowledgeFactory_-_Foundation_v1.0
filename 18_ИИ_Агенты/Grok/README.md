# Grok

Роль агента в Проекте: Используется для быстрого поиска и предварительной сортировки материалов при первичной обработке новых данных.

Промпты для этого агента размещать в `../../07_Промпты` с указанием названия агента в имени файла. Правила использования — см. КП-003 (`../../00_Проект/01_Управление/КП-003_Регламент_работы_ИИ.docx`).
