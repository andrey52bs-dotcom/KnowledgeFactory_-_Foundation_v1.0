# 25_Архив

Устаревшие или заменённые версии документов (см. КП-005, раздел «Архивирование предыдущих версий»).

## Наполнение
**Архив_версий.xlsx** — журнал 9 архивных версий, восстановленных из истории git 11.07.2026.

## Важное примечание (самоаудит)
До 11.07.2026 этот раздел нарушал КП-005: при повышении версии документа старый файл удалялся, а не переносился сюда. Реальная история сохранялась только в git-репозитории, а не в файловой структуре Проекта, как того требует регламент. Обнаружено и исправлено при плановой проверке — 9 версий восстановлены из истории git и размещены здесь с журналом.

Начиная с 11.07.2026 — при повышении версии любого утверждённого документа старая версия должна переноситься в этот раздел (в подпапку с датой архивирования), а не удаляться.
