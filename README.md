# Knowledge Factory «Теплоконтур»

Корпоративная система управления инженерными знаниями: стандарты диагностики, регламенты обслуживания, инструкции, энциклопедия, FAQ, RAG-карточки, промпты для ИИ-агентов, карточки оборудования, каталог неисправностей и аналитика.

**Текущий релиз:** `Release 1.0 — FOUNDATION` (08.07.2026)
**Статус:** учредительная структура и комплект управляющих документов утверждены

---

## Быстрый старт

1. Прочитать [`00_Проект/01_Управление/КП-001_Конституция_проекта.docx`](00_Проект/01_Управление/КП-001_Конституция_проекта.docx) — миссия, принципы и роли Проекта.
2. Ознакомиться с жизненным циклом документа — [`КП-002_Регламент_разработки.docx`](00_Проект/01_Управление/КП-002_Регламент_разработки.docx).
3. Для нового документа — взять шаблон из [`17_Шаблоны`](17_Шаблоны) и следовать [`КП-007_Регламент_управления_документами.docx`](00_Проект/01_Управление/КП-007_Регламент_управления_документами.docx).
4. Актуальный статус всех разделов — [`00_Проект/MANIFEST.xlsx`](00_Проект/MANIFEST.xlsx).

## Структура репозитория

```
00_Проект                    — управление, реестры, версионирование, спринт-отчёты, релизы
01_Стандарты_СТД              — стандарты диагностики (СТД-001 утверждён, v1.0)
02_Регламенты_РГЛ              — регламенты обслуживания
03_Инструкции_ИНС               — инструкции для персонала
04_Энциклопедия                — теория (серия ЭНЦ)
05_FAQ                         — часто задаваемые вопросы
06_RAG                         — RAG-карточки для ИИ-поиска
07_Промпты                     — промпты для ИИ-агентов
08_Карточки_оборудования        — паспортные карточки оборудования
09_Технический_словарь          — единая терминология
10_Каталог_неисправностей        — классификатор неисправностей
11_Физика_процессов              — физические принципы диагностики
12_Типовые_решения               — готовые решения типовых проблем
13_Кейсы_объектов                — практические кейсы
14_Фотоархив / 15_Видеоархив     — медиаматериалы
16_Нормативные_документы         — внешние нормативные документы
17_Шаблоны                       — шаблоны для новых документов
18_ИИ_Агенты                     — материалы по используемым ИИ-агентам
19_Черновики / 20_На_редакции / 21_Утверждено — статусы жизненного цикла документа
22_Word / 23_Markdown / 24_PDF   — экспорт в разных форматах
25_Архив                         — устаревшие версии документов
26_Релизы                        — зафиксированные релизы базы знаний
27_Аналитика                     — аналитика по базе знаний
28_Лаборатория                   — экспериментальные материалы
```

Полное описание назначения каждого раздела — в [`КП-006_Структура_проекта.docx`](00_Проект/01_Управление/КП-006_Структура_проекта.docx).
